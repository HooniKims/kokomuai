# 꼬꼬무AI

꼬꼬무AI는 선생님이 수업용 질문형 AI 챗봇을 만들고, 학생이 공유 링크로 접속해 대화형 학습을 진행할 수 있는 서비스입니다. 챗봇은 정답을 바로 알려주기보다 학생의 생각을 끌어내는 질문을 이어 가며, 선생님은 수업 목표와 질문 수준을 조정해 학습 흐름을 설계할 수 있습니다.

서비스 주소: [https://kokomuai.vercel.app](https://kokomuai.vercel.app)

## 처음 시작하기

1. 선생님 계정으로 회원가입하거나 로그인합니다.
2. 학교를 선택하고, 필요한 경우 관리자 승인을 기다립니다.
3. 챗봇 만들기에서 챗봇 이름과 과목을 입력합니다.
4. 자동으로 채워진 수업 주제, 대화 목표, 페르소나, 질문 수준을 확인하고 필요하면 수정합니다.
5. 챗봇을 저장한 뒤 공유 링크를 학생에게 전달합니다.

## 선생님 사용 방법

### 챗봇 만들기

챗봇 이름과 과목을 먼저 입력하면 수업 주제 카드가 자동으로 작성됩니다. 수업 주제는 교육과정에서 무조건 가져오는 방식이 아니라, 챗봇 이름에 들어 있는 핵심 단어와 과목 정보를 바탕으로 “이 챗봇 이름의 핵심 개념을 이해하게 한다”는 방향으로 제안됩니다.

예를 들어 챗봇 이름에 특정 개념, 단원명, 인물, 현상, 작품명이 들어 있으면 그 단어를 중심으로 학생이 무엇을 이해하면 좋을지 수업 주제가 미리 채워집니다. 자동 작성된 문구는 그대로 써도 되고, 수업 의도에 맞게 직접 고쳐도 됩니다.

### 기본값 확인하기

챗봇 이름과 과목을 입력하면 다음 항목이 미리 작성됩니다.

- 수업 주제: 챗봇 이름의 핵심 단어를 바탕으로 자동 작성
- 대화 목표: 핵심 개념을 학생이 설명할 수 있도록 돕는 방향으로 자동 작성
- 페르소나: 질문형 튜터로 자동 설정
- 질문 수준: 쉽게로 기본 설정

모든 항목은 저장 전에 수정할 수 있습니다. 자동 작성은 초안 역할을 하므로, 선생님은 수업 대상과 활동 방식에 맞게 표현을 다듬으면 됩니다.

### 교육과정 성취기준 사용하기

교육과정 성취기준은 수업 주제를 자동으로 덮어쓰기 위한 필수 조건이 아닙니다. 필요할 때만 성취기준을 선택해 챗봇의 수업 맥락을 보완할 수 있습니다.

성취기준을 고르면 챗봇이 해당 기준을 참고해 대화를 구성합니다. 성취기준을 고르지 않아도 챗봇 이름과 과목을 바탕으로 기본 수업 주제와 대화 목표를 만들 수 있습니다.

### 질문 수준 조정하기

질문 수준은 기본적으로 쉽게로 설정됩니다. 학생 수준이나 수업 의도에 따라 보통 또는 어렵게로 바꿀 수 있습니다.

- 쉽게: 개념을 처음 접하는 학생에게 적합
- 보통: 기본 개념을 알고 있는 학생에게 적합
- 어렵게: 심화 질문이나 토론형 활동에 적합

### 학생에게 공유하기

챗봇을 만든 뒤 생성된 공유 링크를 학생에게 전달합니다. 학생은 별도 로그인 없이 링크로 접속해 대화를 시작할 수 있습니다.

선생님 화면에서는 만든 챗봇을 확인하고, 공유 링크를 복사하거나 필요 없는 챗봇을 삭제할 수 있습니다.

## 학생 사용 방법

1. 선생님이 보낸 챗봇 링크를 엽니다.
2. 이름이나 개인정보를 꼭 입력하지 않아도 대화를 시작할 수 있습니다.
3. 챗봇의 질문에 답하면서 개념을 차근차근 정리합니다.
4. 대화가 끝나면 필요한 경우 PDF 또는 TXT로 저장합니다.

챗봇은 학생의 답을 바탕으로 다음 질문을 이어 갑니다. 기본 질문 수준은 쉽게로 설정되어 있어, 처음 배우는 개념도 부담 없이 시작할 수 있습니다.

## 관리자 사용 방법

관리자는 학교 단위로 선생님 계정과 챗봇 사용 현황을 관리합니다.

- 선생님 계정 승인 또는 비활성화
- 학교별 챗봇 목록 확인
- 최근 활동 로그 확인
- 모델 사용량 확인

## 개인정보와 안전

- 학생은 별도 계정 없이 공유 링크로 참여할 수 있습니다.
- 학생 이름, 전화번호, 주소 같은 개인정보는 입력하지 않는 것이 좋습니다.
- 대화 내용은 수업 활동 확인과 품질 개선을 위해 저장될 수 있습니다.
- 선생님은 공유 링크가 필요한 학생에게만 전달해 주세요.

## 자주 묻는 질문

### 수업 주제가 마음에 들지 않으면 어떻게 하나요?

자동으로 작성된 수업 주제를 직접 수정하면 됩니다. 챗봇 이름과 과목을 다시 고치면 새 주제가 다시 제안될 수 있습니다.

### 교육과정 성취기준을 꼭 선택해야 하나요?

꼭 선택하지 않아도 됩니다. 챗봇 이름과 과목만으로도 기본 수업 주제, 대화 목표, 페르소나가 작성됩니다. 성취기준은 수업 맥락을 더 정확히 맞추고 싶을 때 사용하면 됩니다.

### 챗봇이 너무 어렵게 질문하면 어떻게 하나요?

챗봇 만들기 화면에서 질문 수준을 쉽게로 설정해 주세요. 기본값은 쉽게이며, 필요한 경우 선생님이 보통 또는 어렵게로 바꿀 수 있습니다.

## 사용 권장 흐름

1. 챗봇 이름을 수업 핵심 개념이 드러나도록 정합니다.
2. 과목을 입력합니다.
3. 자동 작성된 수업 주제와 대화 목표를 확인합니다.
4. 학생 수준에 맞게 질문 수준과 힌트 강도를 조정합니다.
5. 챗봇을 저장하고 공유 링크를 전달합니다.

챗봇 이름을 구체적으로 쓸수록 자동 작성되는 수업 주제와 대화 목표도 더 자연스러워집니다.
