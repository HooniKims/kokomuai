# 목표
2022 개정 교육과정의 내용을 반영하여 모든 학교급, 모든 과목에서 사용할 수 있는 챗봇을 만들고 싶다.

# 특징
- 학생이 어떤 과목에서든 질문을 할 수 잇는 챗봇이지만, 절대로 일반 AI 처럼 답변을 주는 것이 아니라 학생의 생각을 확장할 수 있도록 질문을 오히려 하면서 학생 스스로 깨달음을 얻을 수 있어야 한다.
- 학생이 교사가 결정한 과목과 주제에서 벗어난 질문을 하게 되면, 챗봇은 절대로 답변하지 않아야함.

# 챗봇의 페르소나
1. 친절하고 전문적인 선생님
2. 학교급에 맞는 수준으로 질문을 이어가는 전문적인 선생님

# 관련 용어
1. 소크라테스의 산파술
2. 하브루타 교수법
3. Human in the loop
4. 피아제의 인지발달 이론
5. 비고츠키의 사회 문화 이론
6. 존 듀이의 반성적 사고 이론

# 사용할 AI
1. 기본 : ChatGPT 5.4 nano 
2. fallback : gemma4-e4b(우리집 맥미니 로컬 llm 활용)

# 디자인 기준
- 현재 앱에 반영된 `src/presentation/styles.css`를 기준으로 유지한다.
- 초기 디자인 참고용 외부 파일은 정리했으므로 런타임이나 빌드에 의존하지 않는다.

# 웹앱 구조
- 교사는 가입 필요
- 학생들은 가입 불필요
- 교사가 가입하면 자신만의 페이지(대시보드)가 생김
- 교사는 여러개의 챗봇을 만들 수 있음. 학교급, 과목, 챗봇의 주제를 입력하거나 선택할 수 있게 함.
- 교사가 자신만의 페이지를 학생에게 공유하면 학생은 접속해서 챗봇 활동 가능
- 이때, 학생은 각자의 채팅 공간을 가지며, 대화 내용은 로컬에만 저장됨.
- 일반적인 사용 AI 챗봇과 같은 인터페이스 제공. 새대화, 대화기록도 포함할 것.
